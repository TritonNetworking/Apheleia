# Apheleia

A simple, staged, pipeline-oriented dataflow system

## Building

~~~
mkdir build
cd build
cmake ../src
make
~~~

## Running

~~~
cd build && make test
~~~

or

~~~
build/test/testsockets/testsockets
~~~
